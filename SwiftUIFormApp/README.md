# SwiftUIFormApp

This Swift package builds a simple SwiftUI app that mimics a basic Google Form. It runs on iOS (14+) and macOS using Mac Catalyst.

## Building

1. Open `Package.swift` with Xcode 13 or later.
2. Choose an iOS simulator or a Mac (My Mac (Designed for iPad)) scheme.
3. Run the `SwiftUIFormAppApp` scheme (the executable target).

The default admin credentials are:

```
username: admin
password: $uper@dmin
```
